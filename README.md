## Pinout for X2

|         |         |         |         |         |         |         |         |
|---------|---------|---------|---------|---------|---------|---------|---------|
|  CHAN2  |  CHAN3  |  ICOM   |   IX    |  AGND*  |  CHAN4  |  DGND^  |   SCL   |
|  CHAN1  | STIMOUT | STIMIN  |   V-    |   V+    |  AGND*  |  DGND^  |   SDA   |

\*Shorted to each other on board

^Shorted to each other on board
